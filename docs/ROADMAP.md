# Roadmap

## Phase 1 - MVP (Complete)

Core features implemented:

- Authentication (register/login)
- Project management
- Task management with subtasks
- User profiles
- Responsive mobile/desktop design
- Type-safe database with Prisma

## Phase 2 - Usability (Next)

Priority improvements:

- Sections for organization
- Drag-and-drop task reordering
- Search and filtering across tasks
- Keyboard shortcuts
- Loading states and skeletons
- Error handling UI (error boundaries, toast notifications)
- Automated tests (unit, component, integration)

## Phase 3 - Analytics

Add productivity insights:

- Dashboard with task statistics
- Completion trends and charts
- Productivity insights
- Weekly/monthly reports
- Task completion rates
